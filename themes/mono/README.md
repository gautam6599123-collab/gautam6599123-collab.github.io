# Mono - Hugo Theme

**Mono** is a rigid, high-performance, datasheet-inspired theme built for technical writing, mathematics, and portfolio showcases. Designed with a strict minimalist aesthetic, it prioritizes readability, instantaneous load times, and frictionless reading experiences.

## Features

* **Datasheet Aesthetic:** Rigid borders, high-contrast typography, and a technical manual layout.
* **Instantaneous Navigation:** Integrated with `instant.page` for zero-javascript, hover-intent page preloading.
* **Native Math Support:** Built-in KaTeX support for rendering LaTeX block and inline equations globally or per-post.
* **High-Contrast Syntax Highlighting:** Pre-configured with Hugo's native Chroma engine using the `bw` (black-and-white) style.
* **Dynamic Data Pages:** YAML-driven templates for generating structural 'Projects' and 'About' pages without writing HTML.
* **Hyper-Optimized:** Minified CSS/JS asset pipelines and native browser lazy-loading for images.

## Installation
